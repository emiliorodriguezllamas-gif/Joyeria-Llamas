# Manual de Usuario - Joyeria Llamas ERP

Bienvenido a tu nuevo sistema de gestion. Este documento explica como sacar el maximo partido a cada modulo.

## 1. Conceptos Basicos
- **Luxury UI**: El sistema esta diseñado para ser visual. Los botones dorados indican acciones principales.
- **Persistencia**: Todos los datos se guardan en tu navegador. Puedes trabajar sin internet, pero no borres el historial del navegador si no quieres perder los datos (recomendaos hacer backups CSV periodicamente).

## 2. Modulos Principales

### 📦 Inventario
- **Añadir Productos**: Pulsa "Nuevo Articulo". Usa el boton "Auto" para que el sistema genere un SKU (codigo) por ti.
- **Stock Bajo**: Los productos con poco stock parpadearan en rojo para avisarte.
- **Exportar**: Puedes bajar tu catalogo en PDF para enseñarselo a clientes o en CSV para Excel.

### 👥 Clientes
- **Perfil Joyero**: Guarda las tallas de anillos y pulseras de tus clientes para no tener que medirlos cada vez.
- **Alertas**: Si un cliente cumple años o tiene un aniversario en los proximos 30 dias, veras un aviso dorado.

### 🛠️ Taller
- **Tablero Kanban**: Arrastra o cambia el estado de las reparaciones. 
- **Ticket de Resguardo**: Imprime el ticket para el cliente. Es su comprobante para recoger la pieza.

### 🖥️ TPV (Punto de Venta)
- **Venta Rapida**: Haz clic en los servicios (como "Cambio de pila") para añadirlos directo al ticket.
- **IVA**: Puedes quitar el IVA de una venta usando el interruptor dorado en el total.
- **Metodos de Pago**: Soporta Efectivo, Tarjeta y Mixto. Si eliges efectivo, el sistema te dira cuanto cambio devolver.

### 📄 Facturas
- **Control de Ventas**: Mira cuanto has vendido hoy, esta semana o el IVA que llevas recaudado este mes.
- **Re-impresion**: Siempre puedes volver a descargar el PDF de cualquier factura antigua.

## 3. Importacion de Datos
Si vienes de otro sistema:
1. Ve a Inventario o Clientes.
2. Pulsa "Importar CSV".
3. Descarga la **Plantilla**.
4. Rellena tus datos en esa plantilla y subela.
5. El sistema te ayudara a emparejar las columnas.
